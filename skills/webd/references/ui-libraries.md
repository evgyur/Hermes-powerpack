# UI Libraries for Landing Pages

A curated collection of UI libraries built on top of shadcn/ui for creating professional landing pages.

## Component Libraries

### 1. Origin UI
- **Best for:** Consistent design language
- **Components:** Navbars, inputs, calendars
- **Highlights:** Clean code with accessibility built in

### 2. MVP Blocks
- **Best for:** Animated marketing sections
- **Components:** Pricing, waitlist, bento grids
- **Highlights:** Framer Motion animations, insane attention to detail

### 3. Kobalte UI
- **Best for:** Advanced minimalistic components
- **Components:** AI chat, credit cards, video players
- **Highlights:** Sweet spot between clean & functional

### 4. Skipper UI
- **Best for:** Modern animated websites
- **Components:** Slide buttons, card carousels
- **Highlights:** Unique components you won't find elsewhere

### 5. Magic UI
- **Best for:** Landing page sections
- **Components:** 50+ animated components
- **Highlights:** Used by Vercel and Stripe

### 6. Aceternity UI
- **Best for:** 3D animations and effects
- **Components:** Spotlight effects, parallax scrolls
- **Highlights:** Makes your site look expensive

### 7. Cult UI
- **Best for:** SaaS dashboards
- **Components:** Form components
- **Highlights:** Minimal but powerful

### 8. Luxe UI
- **Best for:** Premium feel
- **Components:** Testimonials, features, CTAs
- **Highlights:** Looks like you hired a designer

### 9. Animata
- **Best for:** Copy-paste animations
- **Components:** Loading states, transitions
- **Highlights:** Makes boring pages interesting

### 10. Phosphor Icons
- **Best for:** Iconography
- **Components:** 9000+ icons, 6 weights
- **Highlights:** Alternative to Lucide icons for unique UI

### 11. PatternCraft
- **Best for:** Backgrounds
- **Components:** Patterns and textures
- **Highlights:** Makes empty space interesting, instant visual upgrade

### 12. Tailark
- **Best for:** Startup landing pages
- **Components:** Modern Tailwind components
- **Highlights:** Clean minimal aesthetic

### 13. Smooth UI
- **Best for:** Premium interactions
- **Components:** Hover effects and transitions
- **Highlights:** Buttery smooth animations

### 14. Motion Primitives
- **Best for:** Animation building blocks
- **Components:** Scroll triggers, reveals
- **Highlights:** Framer Motion made easy

### 15. Kokonut UI
- **Best for:** Playful modern designs
- **Components:** Cards, buttons, inputs
- **Highlights:** Stands out from boring defaults

### 16. Bundui
- **Best for:** Fast launches
- **Components:** Hero, features, pricing sections
- **Highlights:** Ready to use sections

### 17. Prism UI
- **Best for:** Glassmorphism
- **Components:** Gradient components
- **Highlights:** Gradients done right, modern aesthetic

## Usage Tips

1. **Mix and match** - Combine libraries for unique results
2. **Don't over-animate** - Use animations purposefully
3. **Check bundle size** - Some animation libraries add weight
4. **Test accessibility** - Ensure motion respects `prefers-reduced-motion`
5. **Stay consistent** - Pick a design language and stick to it
