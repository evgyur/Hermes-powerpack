# Workshop skills bundle

Public-safe skills added for the first three lessons of the new Codex + Hermes workshop.

## Lesson 1 — Codex, GitHub, first workflow

Bundled skills:

- `po` — prompt optimizer for better Codex/Hermes task prompts.
- `perplex` — source-backed current web research.
- `par` — fast breadth-first source discovery.
- `telegram-chip` — safe contract for user-owned Telegram runtime access.
- `gptprof-hermes` — optional GPT/OpenAI-Codex profile switcher with `/gptprof`, `/gptt`, `/mmfast` and Telegram callback support.
- upstream GitHub/Codex skills already included in Hermes: `github-*`, `codex`, `hermes-agent`.

## Lesson 2 — site design with design.md / Refero / Hallmark / Taste / Open Design

Bundled skills:

- `refero-web-design` — reference-led website/UI design workflow.
- `hallmark` — anti-AI-slop design discipline.
- `taste-skill` / `design-taste-frontend` — premium frontend visual polish.
- `image-to-code` — visual implementation workflow for Codex-style site builds.
- `webd` — web design system for landing pages/SaaS/frontend projects.

## Lesson 3 — backend, VPS, Supabase, accounts, CRM, SuperGoal

Bundled skills:

- `supergoal` — principal-grade plan-only execution package.
- `server-doctor` — public-safe VPS/server diagnostics.
- `public-endpoint-ops` — DNS/nginx/TLS/public endpoint verification.
- `porkbun-api-dns` — Porkbun DNS API workflow without committed credentials.
- `supabase-project-ops` — Supabase schema/auth/RLS workflow.

## Privacy boundary

This public repo intentionally excludes private operator hosts, chat IDs, sessions, credentials, organization-specific routes, and production-only deployment details. Real API keys and Telegram sessions must be configured locally by each student.
